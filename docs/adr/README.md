# ADR Compatibility Notice

Architecture Decision Records are maintained in `docs/decisions/`.

## Canonical location

- Use `docs/decisions/README.md` as the entrypoint.
- Create new ADRs in `docs/decisions/` using `YYYY-MM-DD-short-title.md`.

## Why this folder still exists

This directory is kept only for backward compatibility with older links and references.
Do not add new ADR content here.
