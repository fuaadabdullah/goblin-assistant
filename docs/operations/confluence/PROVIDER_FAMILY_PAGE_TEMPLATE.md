# Provider Family Page Template

## Canonical repo sources

- `config/providers.toml`
- `docs/operations/PROVIDERS.md`
- relevant repo/provider docs

## Purpose

Describe what this provider family is used for, how it is configured, and what operators or contributors should know.

## Recommended sections

### What this provider is for

- supported use cases
- notable strengths and constraints

### Required configuration

- env vars
- default model
- provider tier

### Routing and health notes

- known failover behavior
- selectability rules
- circuit or quota caveats

### Jira links

- open provider bugs
- recent provider incidents

### Review metadata

- page owner
- last reviewed date
