# Provider Pricing Reference

## Canonical repo source

- `config/providers.toml`

## Purpose

Show current provider pricing as a reference view for planning and discussion.

## Required statements

- This page is not the source of truth for pricing.
- Pricing values are canonical in `config/providers.toml`.
- Any embedded or pasted table here must be derived from repo data.

## Recommended sections

- per-provider pricing summary
- notable model-specific cost differences
- owner for pricing review
- last refresh date

## Update rule

When pricing changes, update `config/providers.toml` first. Then refresh the Confluence view from repo-derived data.
